# PrimaryCascadeRHEA
The scripts for the article "Impact of chemical short-range ordering and V addition on radiation stability of refractory high entropy alloys" on Computational Materials Science.
MD simulation for the primary cascade of Mo₇₂W₁₃Ta₁₀Ti₂.₅Zr₂.₅ and V₅Mo₆₇W₁₃Ta₁₀Ti₂.₅Zr₂.₅.

https://doi.org/10.1016/j.commatsci.2026.114668.
